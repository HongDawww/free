# 기능
- 할 일을 입력
- 입력 후 enter키를 누르면 입력된 할 일이 출력
- 할 일 완료는 입력된 할 일 앞 버튼을 클릭
- 버튼 클릭시 취소선 생성
- 삭제 기능
- 더블클릭시 해당 할 일 삭제
- 전체 삭제 클릭시 전체 할 일 삭제
- 현재 할 일 목록은 리로드 시에 유지
