# 기능
- 연도, 월 및 날짜 입력할 수 있습니다
- 입력한 날짜까지 남은 시간이 계산되어 확인할 수 있습니다
- 남은 시간은 타이머 기능을 통해 1초마다 변경됩니다
- 입력한 날짜(savedDate)는 저장됩니다
- 초기화 버튼 클릭시 타이머를 종료할 수 있습니다
